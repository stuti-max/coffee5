title - coffee5

description- Many coffee lovers struggle to conveniently discover, browse, and order their favorite coffee products from a single, user-friendly platform, often facing issues like limited menu visibility, lack of customization options, and inefficient ordering processes.
Coffee5 addresses these challenges by providing a seamless digital coffee store experience where users can easily explore a wide variety of beverages, personalize their orders, and place them quickly without confusion or delay.
The app simplifies the entire coffee purchasing journey, saving time, reducing friction, and enhancing customer satisfaction through an intuitive and efficient interface.

framework-Android SDK With Kotlin Plus some Java support

Known Limitations
The Coffee5 application is a functional Android-based coffee store app, but it has some limitations. 
The app depends on Firebase Realtime Database, so it requires a stable internet connection and does not support full offline functionality.
Since it is built using basic Android architecture, it does not include advanced features like role-based admin panels or secure payment gateway integration.

future work-
Offline support can be added using local databases such as Room. A secure payment gateway (e.g., Razorpay or Stripe) can be integrated to enable real purchases.
An admin dashboard can be developed to manage products, orders, and users more efficiently.
The UI can also be improved with more modern animations and better user personalization features like recommendations based on user behavior. Additionally, push notifications and order tracking can be added to improve user engagement and overall experience.
